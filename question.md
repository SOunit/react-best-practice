# 依存関係

## A は B に依存している

- A は B の影響を受ける
- B に変更があれば、A も変更する
- A は B を import する
- 依存元は A, 依存先は B

## UserPage は User に依存している

- UserPage は User の変更の影響を受ける
- User に変更があれば、UserPage も変更する
- UserPage は User を import する
- 依存元は UserPage, 依存先は User
